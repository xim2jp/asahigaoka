# AI動画生成：B-roll・実写風映像・フック

テキスト/画像から実写風・抽象映像を生成するサービス群。**雰囲気の演出・冒頭フック・B-roll**に効くが、**秒課金で高コスト**かつ**画面の正確再現・ブランド統制が苦手**。骨格（Remotion/録画）の"差し色"として最小利用する。

## 使いどころと避けどころ

| 使う | 避ける |
|------|--------|
| ヒーローの冒頭フック（象徴的なシーン） | プロダクト画面の再現（必ず崩れる→録画/Remotion） |
| 抽象イメージ（データが流れる/街/オフィスの空気感） | 正確な文言・数字・ロゴの表示（テロップは Remotion で） |
| SNS広告の"止める"冒頭1–2秒 | 長尺の主役（コストが爆発） |
| 実写撮影が予算的に無理なシーン | 一貫キャラクターの長い物語（破綻しやすい） |

## コスト試算（最重要・野口Profile）

AI動画は**秒（またはクレジット）課金**。塵が積もる。使う前に必ず試算：

```
想定コスト ≈ 生成秒数 × 単価 × 「採用までの試行回数（ボツ込み）」
```

- **試行回数を 3〜10 倍**で見積もる（一発で理想は出ない。プロンプト調整で何度も生成し直す）
- → 「5秒のフックを採用」でも実際は 30〜50秒分生成している、というのが普通
- **対策**：尺を短く（フックは2–3秒で十分）、本数を絞る、画像→動画（image-to-video）で当たりを安定させる、安価サービスで当たりを探し本番だけ高品質サービス

> **判断基準**：AI動画の月額/案件コストが**クライアント費用負担で賄えるか、または高ROIが明確か**。Tier A の自社MVPでは原則フック数秒に限定。

## プロンプトの型

```
[被写体] + [動作] + [カメラ（dolly in / aerial / close-up）]
+ [画調（cinematic / soft light / 35mm）] + [色トーン（brand: indigo 基調）]
+ [尺感・テンポ] ＋ [ネガティブ：text, watermark, distorted hands]
```

- ブランドカラーを画調プロンプトに含めて Remotion 部分と馴染ませる
- 文字は入れさせない（崩れる）。テロップは Remotion
- image-to-video：先に AI画像（visual-director のツール）で構図を固め、それを動かすと安定

## 権利・コンプラ（必ず確認）

- **商用利用可否**：プランごとに異なる。営業利用は商用可プラン必須
- **透かし（watermark）**：無料/下位プランは透かし入りのことがある
- **実在人物・ブランド・著作物**：他社ロゴ・有名人・既存キャラを生成しない
- **AI生成の開示**：媒体・広告審査によっては表示が必要になる場合がある
- **学習・帰属**：生成物の権利方針はサービス規約に従う

## ハイブリッド運用フロー

```
1. video-strategy.md で「AIに任せるカット」を絵コンテで1–2カットに限定
2. 安価サービスでプロンプト探索 → 当たりを image-to-video で安定生成
3. 採用クリップを Remotion / 編集に取り込み、テロップ・ブランド・CTAを後載せ
4. 録画(本物のUI) と組み合わせて1本に
```

## チェックリスト

- [ ] AI動画は冒頭フック/B-roll に限定（画面再現に使っていない）
- [ ] 生成秒数 ×単価 ×試行回数（3–10倍）でコスト試算した
- [ ] 商用利用可プラン・透かしなしを確認
- [ ] 実在人物/他社ブランド/著作物を生成していない
- [ ] テロップ・数字・ロゴは Remotion で後載せ
- [ ] ブランドカラー/画調を Remotion 部分と馴染ませた
