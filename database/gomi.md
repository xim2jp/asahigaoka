# 練馬区旭丘一丁目 ゴミ・資源の出し方

## 収集曜日一覧（令和7年度）

旭丘は全域で以下の曜日に収集されます。

| 種別 | 収集曜日 |
|-----|---------|
| 可燃ごみ | **水曜日・土曜日**（週2回） |
| 不燃ごみ | **第1・第3月曜日**（月2回） |
| 容器包装プラスチック | **火曜日**（週1回） |
| 古紙（新聞・ダンボール等） | **火曜日**（週1回） |
| びん・缶 | **火曜日**（週1回） |
| ペットボトル | **火曜日**（週1回） |

## 出す時間

- **朝8時まで**に出してください
- 収集は朝8時から順次行われます
- 前日の夜に出すのは禁止です（カラスや猫による被害防止のため）

## ゴミの分別方法

### 可燃ごみ（水曜・土曜）
- 生ごみ（よく水を切って）
- 紙くず、木くず
- 衣類（資源にならないもの）
- ゴム・皮革製品
- プラスチック製品（容器包装以外）

### 不燃ごみ（第1・3月曜）
- 金属類（なべ・やかん以外）
- ガラス製品
- 陶器
- 小型家電（30cm未満）
- 蛍光灯・電球
- スプレー缶（穴を開けずに出せます）
- モバイルバッテリー、ハンディファン、加熱式たばこ

### 容器包装プラスチック（火曜）
- 「プラ」マークのついた容器・包装
- レジ袋、ポリ袋
- 食品トレイ
- カップ麺・弁当の容器
- ボトル類のキャップ
- 発泡スチロール

※汚れがひどいものは可燃ごみへ

### 古紙（火曜）
- 新聞紙（チラシを含む）
- 雑誌・書籍
- ダンボール
- 紙パック（牛乳パック等）
- 雑がみ（菓子箱、包装紙など）

※ひもで十字にしばって出す

### びん・缶（火曜）
- 飲食用のびん（ジュース、調味料など）
- 飲食用の缶（ジュース、缶詰など）

※軽くすすいで出す
※ふたは外す（金属ふたは不燃ごみ、プラふたは容器包装プラスチック）

### ペットボトル（火曜）
- 「PET」マークのついたボトル

※キャップとラベルは外して容器包装プラスチックへ
※軽くすすいで出す
※つぶして出す

## 粗大ごみ（有料・申込制）

一辺が30cm以上のものは粗大ごみです。

### 申込方法
1. **インターネット**: 練馬区粗大ごみ受付センター（24時間受付）
2. **電話**: 03-5703-5399（月〜土 8:00〜19:00）

### 手数料
- 品目によって400円〜2,800円
- 粗大ごみ処理券（シール）をコンビニ等で購入
- 処理券を貼って、指定日の朝8時までに出す

## 区では収集できないもの

以下は区では収集できません。専門業者や販売店に相談してください。

- **家電4品目**: エアコン、テレビ、冷蔵庫・冷凍庫、洗濯機・衣類乾燥機
- **パソコン**: メーカー回収または小型家電回収ボックス
- **バッテリー（車・バイク用）**
- **タイヤ**
- **消火器**
- **ガスボンベ（プロパン）**
- **ピアノ**
- **耐火金庫**
- **灯油、ペンキ、農薬などの液体・危険物**
