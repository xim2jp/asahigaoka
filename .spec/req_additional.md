# 機能追加依頼その１
管理画面 /admin/login.html から、**スマホでログインした場合**スマホ専用の管理画面に遷移させたい

## スマホ版管理画面
スマホで見やすいようなサイズで以下の機能を実装
1. 記事一覧
 - 現在公開中の記事が時系列に30件まで並ぶ。31件以上古いものは対象外とする
 - 件名・本文のみ編集ができる
 - 下書き／公開中のスイッチができる　下書きから公開中に変更した場合、まだ一度もその記事をSNSに投稿していなければ自動投稿する
 
2. 記事フォーム
 以下の項目のみのシンプルな記事フォームを提供する
 - 日付範囲（Fromのみ必須）カレンダー機能実装
 - 件名
 - 要約
 - アイキャッチ画像
 - 「AIに依頼」ボタン
 - 本文
 - SNS用サマリ文
 - 投稿・キャンセルボタン
 PC版にもある、件名、日付、本文を入力するとAIに清書を依頼し、本文SNS用サマリ文が自動で埋まる
 保存は自動で下書きステータスにする。
